Ethics Analysis

This page identifies and considers how to handle some of the ethical risks associated with a newsletter plug-in.

ACS Framework: trustworthiness

Risk: Exposure of sensitive personal data through insecure storage or unauthorized access.

Proposed solutions

  • Ensure secure encryption and storage of data
  • Conduct tests on the plug-in to determine its security

Broader considerations

  • Limit the amount of data stored over the whole website

ACS Framework: Honesty

Risk: Demographic data could enable profiling or exclusion in resource allocation. This would be discrimination.

Proposed solutions

  • Anonymize data
  • Provide opt out options

Broader considerations

  • Create a page that demonstrates transparancey on how the data is used

Framework: Respect

Risk: Invasive data requests could deter vulnerable groups from accessing online resources.

Proposed solutions

  • Reduce the presented fields
  • Provide an offline subscription option

Broader considerations

  • Ensure the remainder of the website maximises accessibility for vulnerable groups.
