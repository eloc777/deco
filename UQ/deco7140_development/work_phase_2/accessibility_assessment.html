Accessibility assessment

This page lists the relevant accesibility success criteria. For each criteria, there is an explanation for why it is relevant, as well as listed features which help the website achieve this success criteria

Relevant Success criteria

1.1.1 Non-text content

Relevance

For cultural and heritage communities, where cultural imagery may play a significant role, providing meaningful alt text ensures that everyone can access and understand the significance of visuals

Feature

The website uses aria methods which allow screen readers to understand where they are on the website. Moving forward it will be important to provide text descriptions of any images or non textual content.

1.4.1 Avoid Color-only Cues

Relevance

Relying solely on color to convey information can exclude users with color blindness or other visual impairments. This is important to address to maintain the inclusivity value.

Feature

The website avoids any Color-only Cues

1.4.3 Perceivable Text Contrast

Relevance

Keeping with the theme of inclusivity, percievable text contrast makes it easier for all too read.

Feature

The set of use of base colours ensures all contrasts achieve an acceptable ratio.

1.4.4 Resize Text without Loss

Relevance

Given the strong feedback regarding Elder usability, it is especially important that text can be resized without punishing user experience.

Feature

We use css such as flexbox to ensure all textual aspects of the page are dyanimcally resizable.

2.1.1 Keyboard Navigability

Relevance

Again relating to inclusivity, we want all members of the community to be able to navigate the website.

Feature

Featurs such as these flip cards are all accesible through use of the tabindex.

3.1.5 Reading Level

Relevance

With the website's target audience varying greatly in range and education level, a simple reading level is vital.

Feature

The website avoids the use of complicated words. Moving forward, sentence and paragraph structure for crucial pages will be simplified even further.

3.2.3 Consistent Navigation

Relevance

Consistent navigation will make it easier for elders and children alike to not feel so overwhelmed with the website.

Feature

The website includes a consistent menu layout across every page.

3.3.2 Labels (& Instructions)

Relevance

In keeping with the theme of ease of use, labels and instructions will provide a more friendly user experience. This makes it more likely for members to continue using the website.

Feature

All pages of the website contain clear definitions of what is found on that page.
